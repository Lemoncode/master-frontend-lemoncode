# CSS Animations

Esta _masterclass_ dedicada a animaciones con CSS gira entorno al _playground_ que encontraréis en la carpeta `02-playground-animations`.

## Instrucciones

- Copiar o clonar `02-playground-animations` a vuestra carpeta destino.
- Abrimos el proyecto con `VSCode` y desde la terminal instalamos las dependencias ejecutando:

  ```text
  npm install
  ```

- Y a continuación lo arrancamos mediante:

  ```text
  npm start
  ```

## Trabajando con el _playground_

El proyecto es estático y simple, con páginas HTML tradicionales.

Cada ejercicio tiene 2 páginas, una para practicar y otra con la solución.

Para la mayoría de ejercicios encontrarás guias `README.md` en la carpeta de su página correspondiente.
