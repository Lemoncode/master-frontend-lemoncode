# 03 Login

In this sample we are going to create a `login` page.

We will take a startup point sample _02 Properties_.

Summary steps:

- Delete `Hello.vue`.
- Update `App.vue`.
- Update `index.html`.
- Create `login` page.
- Configure router navigation.
- Create `recipe list` page.
- Create `LoginEntity` model.
- Create fake `login` API.
- Check valid login.
